# intern-test
Repository for Intern Test Suitmedia MSIB 5. Built with Kotlin.

## Tech Stack Used:
- Room database
- Paging 3
- RemoteMediator
- ViewModel
- Retrofit
- RecyclerView
- DiffUtil
- CustomView
